## 🚀 Как активировать

1. Нажмите правой кнопкой на **XOLIRX_ACTIVATION.bat**
2. Выберите **"Запуск от имени администратора"**
3. Готово!

## 🔄 Восстановление

Запустите **XOLIRX_RESTORE.bat** от администратора
